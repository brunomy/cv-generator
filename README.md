# Editor A4 em HTML (React + Vite)

Aplicação React para editar um HTML de currículo e visualizar em uma folha A4, com exportação para PDF.

## Como rodar

```bash
npm install
npm run dev
```

## Como editar o template

- Edite o arquivo `public/template.html`.
- A aplicação carrega esse arquivo ao abrir e quando você clica em **Resetar modelo**.

## Exportar para PDF

- Clique em **Baixar PDF** para gerar o arquivo.

## Observações

- A prévia usa tamanho A4 (210 × 297 mm).
- A impressão do navegador também funciona se preferir.
